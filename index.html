Mau Belajar Apa Hari Ini

HTML

HTML (Hypertext Markup Language) adalah bahasa markup untuk membuat halaman web menggunakan simbol tertentu (tag) yang nantinya akan diterjemahkan oleh browser ke halaman web. Dengan menggunakan tag-tag HTML.

Selengkapnya
CSS

CSS adalah singktan dari cascading style sheets, yaitu bahasa yang digunakan untuk menentukan tampilan dan formar halaman website. Dengan CSS, Anda bisa mengatur jenis font, warna tulisan, dan latar belakang halaman.

Selengkapnya
JS

JavaScript adalah bahasa pemrograman populer yang digunakan untuk membuat situs degan konten website yang dinamis artinya konten dapat bergerak atau berubah di depan layar tanpa perlu mereload halaman.

Selengkapnya
Framework Bootstrap

Bootstrap adalah framework HTML,CSS,danJavaScript yang berfungsi untuk mendesain website responsive dengan cepat dan mudah. Framework ini tersusun dari kumpulan file CSS dan JavaScript berbentuk class yang tinggal pakai.

Selengkapnya
PHP

PHP (PHP: Hypertext Preprocessor) adalah sebuah bahasa pemrograman server side scripting yang bersifat open source. PHP menjalankan instruksi pemrograman saat proses runtime. Hasil dari instruksi tentu akan berbeda tergantung data yang diproses.

Selengkapnya
Laravel

Laravel adalah framework berbasis bahasa pemrogramanPHP yang bisa digunakan untuk membantu proses pengembangan sebuah website agar lebih maksimal. Dengan menggunakan Laravel, website yang dihasilkan akan lebih dinamis.

Selengkapnya
